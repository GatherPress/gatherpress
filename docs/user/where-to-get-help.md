# Where to get help

If you need help with GatherPress, there are several official and community-supported channels.

## Official documentation

* User documentation: <https://gatherpress.org/documentation/>  
* Developer documentation: <https://gatherpress.org/documentation/> (developer section)
* On GitHub, on the Main branch [/docs](https://github.com/GatherPress/gatherpress/tree/main/docs)

## Sensitive and safety issues

Report sensitive issues to plugin@gatherpress.org and follow the [Security Policy](https://github.com/GatherPress/gatherpress/blob/develop/SECURITY.md). 

## GitHub

Report bugs and request features: <https://github.com/GatherPress/gatherpress/issues>

When reporting an issue, include:

* Your WordPress version  
* Your GatherPress version  
* Clear steps to reproduce the problem

## Community and discussions

* Join the GatherPress community on the [GatherPress Slack](https://join.slack.com/t/gatherpress/shared_invite/zt-2luaqcruf-iQm_o2UuKBpnX7zfRCxMAg)  
* The [GatherPress channel](https://wordpress.slack.com/archives/C07NB4N0ESJ) on the main WordPress Slack

Community channels are the best place to:

* Ask usage questions  
* Share feedback  
* Discuss future improvements
